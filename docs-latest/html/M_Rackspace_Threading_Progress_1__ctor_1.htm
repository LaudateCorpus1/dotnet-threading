handler
Type: SystemActionT
A handler to invoke for each reported progress value. This handler will be invoked in addition to any delegates registered with the ProgressChanged event. Depending on the SynchronizationContext instance captured by the ProgressT at construction, it is possible that this handler instance could be invoked concurrently with itself.
